## Usage

2. **Development:**

* To install node_modules:

   * backend:
   ```bash
      cd backend
      npm install
   ```

  * frontend: 

   ```bash
      cd frontend
      npm install
      ```
* To run the application in development mode:

    * backend:
   ```bash
      cd backend
      npm run start
   ```

  * frontend: 

   ```bash
      cd frontend
      npm run dev
   ```

**Production:**

* To build and run the production version:

   * backend:
   ```bash
      cd backend
      npm run build
   ```

  * frontend: 

   ```bash
      cd frontend
      npm run build
   ```
